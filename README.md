## Usage

1. Clone this repo

2. Install dependancies

   ```
   yarn
   ```

3. Review outdated dependancies and upgrade if needed

   for eslint, airbnb config

   ```
   npx install-peerdeps --dev eslint-config-airbnb
   ```

   ```
   yarn outdated
   ```

4. Run app

   ```
   yarn dev
   ```

   or if you have now-cli installed,

   ```
   now dev
   ```
